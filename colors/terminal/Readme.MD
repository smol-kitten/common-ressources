# Terminal Color Themes

10 curated terminal color themes in a structured JSON format — with ready-to-use exports for Windows Terminal and Alacritty, and a PHP script that renders all themes as a visual preview PNG.

## Files

| File | Description |
|---|---|
| [`themes.json`](themes.json) | Canonical data — 10 themes with full 16-color palettes |
| [`colortest.php`](colortest.php) | PHP renderer — generates a terminal-window preview PNG |
| [`export/windows-terminal.json`](export/windows-terminal.json) | Windows Terminal color schemes array |
| [`export/alacritty.toml`](export/alacritty.toml) | Alacritty TOML config (Dracula active, others commented) |

The 8 color keys follow the ANSI standard order. Each has a normal and a `bright-` variant.

## Using the exports

### Windows Terminal

Open your `settings.json` → `schemes` array and paste in any entry from `export/windows-terminal.json`. Then set it as the profile's `colorScheme`.

```json
"schemes": [
  { "name": "Dracula", "background": "#282A36", ... }
],
"profiles": {
  "defaults": { "colorScheme": "Dracula" }
}
```

### Alacritty

The `export/alacritty.toml` file has Dracula active. To switch theme, replace the `[colors.*]` blocks with one of the commented alternatives, or point Alacritty's `import` to this file:

```toml
# alacritty.toml
import = ["/path/to/colors/terminal/export/alacritty.toml"]
```

## Examples

### Shell — list all theme backgrounds

```bash
jq '.[] | "\(.name): \(.background)"' themes.json
```

### PHP — look up a theme by slug

```php
$themes = json_decode(file_get_contents('themes.json'), true);
$bySlug = array_column($themes, null, 'slug');
$dracula = $bySlug['dracula'];
echo $dracula['colors']['green']; // #50FA7B
```

### JavaScript — build a CSS block from a theme

```javascript
const themes = await fetch('themes.json').then(r => r.json());
const theme = themes.find(t => t.slug === 'nord');
const css = `
  :root {
    --term-bg: ${theme.background};
    --term-fg: ${theme.foreground};
    --term-red: ${theme.colors.red};
    --term-green: ${theme.colors.green};
    --term-blue: ${theme.colors.blue};
  }
`;
```
