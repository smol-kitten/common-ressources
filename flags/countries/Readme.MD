# Country Flags

Simple stripe-based national flags for countries worldwide. Only flags that can be fully represented using horizontal or vertical stripes are included — this excludes flags with complex emblems, crosses, or non-stripe geometry.

## JSON

### Structure as of 23.05.2026

```json
{
  "name": "Germany",
  "iso": "DE",
  "colors": ["#000000", "#DD0000", "#FFCE00"],
  "flag-version": "common",
  "type": "horizontal-stripes",
  "description": "Description of the flag and its colors.",
  "source": "URL to Wikipedia or official source",
  "continent": "Europe"
}
```

### Fields

| Field | Type | Description |
|---|---|---|
| `name` | string | Country name in English |
| `iso` | string | ISO 3166-1 alpha-2 country code |
| `colors` | string[] | Hex color values in stripe order (top-to-bottom or left-to-right) |
| `flag-version` | string | `common` (civil flag), `state`, `war`, etc. |
| `type` | string | `horizontal-stripes`, `vertical-stripes`, or `[0-360]-degree-stripes` |
| `description` | string | Background on the flag's history and symbolism |
| `source` | string | Source URL |
| `continent` | string | `Europe`, `Africa`, `Americas`, `Asia`, `Oceania` |

### Stripe Types

- `horizontal-stripes` — stripes run left to right, stacked top to bottom (Germany, Austria, Ukraine...)
- `vertical-stripes` — stripes run top to bottom, side by side (France, Italy, Ireland...)

### Coverage

Currently includes ~23 countries across Europe, Africa, and the Americas. Only simple-stripe flags are included — countries with complex heraldic elements (e.g., Mexico, Brazil) are not present.

## Example

```php
$flags = json_decode(file_get_contents('flags.json'), true);
foreach ($flags as $flag) {
    echo $flag['name'] . ' (' . $flag['iso'] . '): ' . implode(', ', $flag['colors']) . "\n";
}
```
